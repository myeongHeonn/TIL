# TypeScript 학습 저장소

## 개요
이 폴더는 TypeScript 개념을 체계적으로 학습하고 정리하는 공간입니다. Claude Code와 협력하여 학습 자료를 작성합니다.

## 파일 구조
- 각 TypeScript 개념별로 별도의 Markdown 파일로 관리합니다.
- `README.md`: 폴더의 인덱스 및 학습 가이드 (수정 금지)
- `CLAUDE.md`: 이 파일 (협업 규칙)
- 개념 문서: 주제별 학습 파일들

## 파일 네이밍 규칙
- 파일명: 공백 사용, 한글로 작성 (예: `01 기본 개념 및 장점.md`)
- 넘버링: 학습 순서가 있는 개념은 앞에 번호 붙임 (예: `01`, `02`, ...)
- 가이드 문서: 순서가 없는 경우 넘버링 생략 (예: `타입스크립트 실행.md`)
- 금지: 언더바(`_`), 대시(`-`)는 사용하지 않기

## 문서 작성 스타일
- **자세함**: 개념을 완전하고 깊이 있게 설명합니다.
- **가독성**: 마크다운 형식을 활용해 명확하고 정돈된 구조를 유지합니다.
- **코드 예제**: 컴파일 에러 vs 올바른 사용법을 대비하여 보여줍니다.
- **강조**: 핵심 개념은 **굵은 글씨** 또는 `코드하이라이트`로 표시합니다.
- **한글**: 모든 설명과 제목은 한글로 통일합니다.

## 작성 진행 상황

### 완료된 문서
- `02 변수 선언과 스코프.md` - var/let/const 차이, 스코프, 호이스팅, 타입 어노테이션, 네이밍 컨벤션

### 작성 예정 문서
- `03 기본 타입` - string, number, boolean, null, undefined, any, unknown, never
- `04 고급 타입` - Union, Intersection, Literal, Enum 등
- `05 함수` - 함수 선언, 매개변수 타입, 반환 타입, 화살표 함수, 오버로딩
- `06 인터페이스` - 인터페이스 정의, 확장, 선택적 속성, 읽기 전용 속성
- `07 클래스` - 클래스 정의, 생성자, 접근 제어자, 상속, 추상 클래스
- `08 제네릭` - 제네릭 함수, 클래스, 인터페이스, 제약조건
- `09 데코레이터` - 클래스 데코레이터, 메서드 데코레이터, 속성 데코레이터
- `타입스크립트 실행` - 환경 설정, 컴파일, 실행 방법

## Git 워크플로우
- 문서 작성 후 **유저가 명시적으로 요청할 때만** commit/push를 수행합니다.
- Commit 메시지 규칙:
  - 신규 추가: `docs: add [개념 이름]`
  - 기존 수정: `docs: update [개념 이름]`
