# 배열

## 개요

배열은 **같은 타입의 값들을 순서대로 저장하는 자료구조**입니다. TypeScript에서는 배열의 요소 타입을 명시적으로 지정하여 타입 안정성을 보장합니다. 이 장에서는 배열 선언 방식, 타입 지정, 자주 사용되는 배열 메서드, 그리고 제네릭을 활용한 고급 배열 사용법을 학습합니다.

---

## 1. 배열 선언

### 1-1. 기본 배열 선언

**타입 뒤에 대괄호 추가하는 방식** (권장)

```typescript
// 문자열 배열
const fruits: string[] = ["apple", "banana", "orange"];

// 숫자 배열
const numbers: number[] = [1, 2, 3, 4, 5];

// 불린 배열
const flags: boolean[] = [true, false, true];

// 빈 배열 (초기값 없음)
const emptyArray: number[] = [];
```

**Array<T> 제네릭 방식**

```typescript
// string[] 과 동일
const fruits: Array<string> = ["apple", "banana"];

// number[] 과 동일
const numbers: Array<number> = [1, 2, 3];
```

**타입 추론 (타입 명시 생략)**

TypeScript는 초기값으로부터 자동으로 배열의 타입을 추론합니다:

```typescript
const fruits = ["apple", "banana"];  // string[] 로 추론
const numbers = [1, 2, 3];           // number[] 로 추론
const mixed = [1, "two", true];      // (number | string | boolean)[] 로 추론
```

---

### 1-2. 여러 타입을 담는 배열 (Union 타입)

배열이 여러 타입의 값을 가질 수 있도록 선언할 수 있습니다:

```typescript
// 숫자 또는 문자열을 담을 수 있는 배열
const mixed: (number | string)[] = [1, "two", 3, "four"];

// 숫자 또는 불린을 담을 수 있는 배열
const booleanOrNumber: (boolean | number)[] = [true, 42, false];

// ❌ 타입이 맞지 않으면 에러 발생
mixed.push(true);  // Error: Argument of type 'boolean' is not assignable to parameter of type 'string | number'
```

---

### 1-3. 객체의 배열

객체를 요소로 하는 배열을 선언할 수 있습니다:

```typescript
// 객체 구조를 인라인으로 정의
const users: { name: string; age: number }[] = [
  { name: "Alice", age: 25 },
  { name: "Bob", age: 30 },
  { name: "Charlie", age: 35 }
];

// ❌ 속성이 없으면 에러
const invalidUser: { name: string; age: number }[] = [
  { name: "Dave" }  // Error: Property 'age' is missing
];
```

인터페이스나 타입 별칭을 활용한 방식 (권장):

```typescript
interface User {
  name: string;
  age: number;
}

const users: User[] = [
  { name: "Alice", age: 25 },
  { name: "Bob", age: 30 }
];
```

---

### 1-4. 읽기 전용 배열 (readonly)

배열의 요소를 변경할 수 없도록 선언할 수 있습니다:

```typescript
// 읽기 전용 배열 선언
const readonlyNumbers: readonly number[] = [1, 2, 3];

// ❌ 요소 수정 불가
readonlyNumbers[0] = 10;  // Error: Index signature in type 'readonly number[]' only permits reading

// ❌ 배열 메서드로 수정 불가
readonlyNumbers.push(4);   // Error: Property 'push' does not exist on type 'readonly number[]'

// ✅ 읽기는 가능
console.log(readonlyNumbers[0]);  // 1
console.log(readonlyNumbers.length);  // 3
```

ReadonlyArray 제네릭 방식도 동일합니다:

```typescript
const readonlyStrings: ReadonlyArray<string> = ["a", "b", "c"];
```

---

### 1-5. 튜플 (Tuple) - 길이와 타입이 고정된 배열

**튜플**은 **정확한 길이와 각 위치의 타입이 정해진 특수한 배열**입니다:

```typescript
// 2개 요소: 첫 번째는 string, 두 번째는 number
const pair: [string, number] = ["age", 25];

// ✅ 올바른 접근
console.log(pair[0]);  // "age" (string)
console.log(pair[1]);  // 25 (number)

// ❌ 잘못된 타입
const wrongPair: [string, number] = [25, "age"];  // Error: Type 'number' is not assignable to type 'string'

// ❌ 길이가 맞지 않음
const incompletePair: [string, number] = ["age"];  // Error: Type '["age"]' is not assignable to type '[string, number]'
```

**선택적 요소를 가진 튜플:**

```typescript
// 2번째 요소는 선택사항
const optionalTuple: [string, number?] = ["name"];    // ✅ OK
const optionalTuple2: [string, number?] = ["name", 30];  // ✅ OK

// ❌ 3번째 요소는 불가
const invalid: [string, number?] = ["name", 30, true];
```

**가변 길이 튜플:**

```typescript
// 첫 번째는 string, 나머지는 모두 number
const variableTuple: [string, ...number[]] = ["score", 10, 20, 30];

// ✅ OK
const variableTuple2: [string, ...number[]] = ["data"];
const variableTuple3: [string, ...number[]] = ["data", 1, 2, 3, 4, 5];

// ❌ 잘못된 타입
const invalid: [string, ...number[]] = ["data", 1, "two"];  // Error
```

## 2. 배열 메서드

### 2-1. 요소 접근 및 확인

**길이 확인**

```typescript
const numbers = [10, 20, 30];
console.log(numbers.length);  // 3

// 마지막 요소 접근
console.log(numbers[numbers.length - 1]);  // 30
```

**특정 요소 확인**

```typescript
const fruits = ["apple", "banana", "orange"];

// includes: 요소 포함 여부
console.log(fruits.includes("apple"));   // true
console.log(fruits.includes("grape"));   // false

// indexOf: 요소의 인덱스 (없으면 -1)
console.log(fruits.indexOf("banana"));   // 1
console.log(fruits.indexOf("grape"));    // -1

// find: 조건을 만족하는 첫 번째 요소
const numbers = [10, 20, 30, 40];
const found = numbers.find(n => n > 25);
console.log(found);  // 30

// findIndex: 조건을 만족하는 첫 번째 요소의 인덱스
const index = numbers.findIndex(n => n > 25);
console.log(index);  // 2
```

### 2-2. 배열 변경 (뮤테이션)

**push: 끝에 요소 추가**

```typescript
const numbers: number[] = [1, 2, 3];
numbers.push(4);
console.log(numbers);  // [1, 2, 3, 4]

// ❌ const 배열도 내용 수정 가능
const constArray: number[] = [1, 2, 3];
constArray.push(4);  // ✅ OK (배열 자체의 재할당이 아님)
constArray = [5, 6];  // ❌ Error (배열 재할당)
```

**unshift: 처음에 요소 추가**

```typescript
const fruits = ["apple", "banana"];
fruits.unshift("grape");
console.log(fruits);  // ["grape", "apple", "banana"]
```

**pop: 마지막 요소 제거**

```typescript
const numbers = [1, 2, 3];
const removed = numbers.pop();
console.log(removed);  // 3
console.log(numbers);  // [1, 2]

// 빈 배열에서 pop
const empty: number[] = [];
console.log(empty.pop());  // undefined
```

**shift: 첫 번째 요소 제거**

```typescript
const fruits = ["apple", "banana", "orange"];
const first = fruits.shift();
console.log(first);  // "apple"
console.log(fruits);  // ["banana", "orange"]
```

**splice: 특정 위치에서 요소 추가/제거**

```typescript
const numbers = [1, 2, 3, 4, 5];

// 인덱스 2부터 2개 요소 제거
numbers.splice(2, 2);
console.log(numbers);  // [1, 2, 5]

// 인덱스 1부터 0개 제거하고 "a", "b" 추가
const letters = ["x", "y", "z"];
letters.splice(1, 0, "a", "b");
console.log(letters);  // ["x", "a", "b", "y", "z"]

// 제거된 요소들을 반환
const items = [10, 20, 30, 40];
const removed = items.splice(1, 2);
console.log(removed);  // [20, 30]
console.log(items);    // [10, 40]
```

**sort: 배열 정렬**

```typescript
const numbers = [3, 1, 4, 1, 5];
numbers.sort();
console.log(numbers);  // [1, 1, 3, 4, 5]

// 문자열 정렬
const fruits = ["banana", "apple", "cherry"];
fruits.sort();
console.log(fruits);  // ["apple", "banana", "cherry"]

// 비교 함수로 커스텀 정렬
const nums = [3, 1, 4, 1, 5];
nums.sort((a, b) => b - a);  // 내림차순
console.log(nums);  // [5, 4, 3, 1, 1]

// 객체 배열 정렬
interface User {
  name: string;
  age: number;
}

const users: User[] = [
  { name: "Alice", age: 30 },
  { name: "Bob", age: 25 },
  { name: "Charlie", age: 35 }
];

users.sort((a, b) => a.age - b.age);  // 나이 오름차순
console.log(users);
```

**reverse: 배열 역순 정렬**

```typescript
const numbers = [1, 2, 3, 4, 5];
numbers.reverse();
console.log(numbers);  // [5, 4, 3, 2, 1]
```

### 2-3. 배열 변환 (Non-mutating)

**map: 각 요소를 변환하여 새 배열 생성**

```typescript
const numbers = [1, 2, 3, 4];

// 각 요소를 2배로
const doubled = numbers.map(n => n * 2);
console.log(doubled);  // [2, 4, 6, 8]

// 숫자를 문자열로 변환
const strings = numbers.map(n => `number: ${n}`);
console.log(strings);  // ["number: 1", "number: 2", ...]

// 객체 배열 변환
interface Product {
  name: string;
  price: number;
}

const products: Product[] = [
  { name: "Apple", price: 100 },
  { name: "Banana", price: 50 }
];

const names = products.map(p => p.name);
console.log(names);  // ["Apple", "Banana"]

// 타입 변환
const result = numbers.map(n => n.toString());
// result의 타입은 string[]
```

**filter: 조건을 만족하는 요소만 선택**

```typescript
const numbers = [1, 2, 3, 4, 5, 6];

// 짝수만 선택
const evens = numbers.filter(n => n % 2 === 0);
console.log(evens);  // [2, 4, 6]

// 객체 배열 필터링
interface User {
  name: string;
  active: boolean;
}

const users: User[] = [
  { name: "Alice", active: true },
  { name: "Bob", active: false },
  { name: "Charlie", active: true }
];

const activeUsers = users.filter(u => u.active);
console.log(activeUsers);  // [{ name: "Alice", ... }, { name: "Charlie", ... }]
```

**reduce: 배열을 단일 값으로 축약**

```typescript
const numbers = [1, 2, 3, 4, 5];

// 합계
const sum = numbers.reduce((acc, curr) => acc + curr, 0);
console.log(sum);  // 15

// 곱셈
const product = numbers.reduce((acc, curr) => acc * curr, 1);
console.log(product);  // 120

// 객체 생성
const items = ["apple", "banana", "apple", "cherry", "banana"];
const count = items.reduce((acc, item) => {
  acc[item] = (acc[item] || 0) + 1;
  return acc;
}, {} as Record<string, number>);
console.log(count);  // { apple: 2, banana: 2, cherry: 1 }
```

**forEach: 각 요소에 대해 함수 실행 (반환값 없음)**

```typescript
const fruits = ["apple", "banana", "orange"];

// 각 요소 출력
fruits.forEach(fruit => {
  console.log(fruit);
});
// 출력:
// apple
// banana
// orange

// 인덱스와 함께 순회
fruits.forEach((fruit, index) => {
  console.log(`${index}: ${fruit}`);
});
// 출력:
// 0: apple
// 1: banana
// 2: orange
```

**join: 배열을 문자열로 변환**

```typescript
const fruits = ["apple", "banana", "orange"];

// 기본 구분자 (쉼표)
console.log(fruits.join());  // "apple,banana,orange"

// 커스텀 구분자
console.log(fruits.join(" | "));  // "apple | banana | orange"
console.log(fruits.join(", "));   // "apple, banana, orange"
console.log(fruits.join(""));     // "applebananaorange"
```

**slice: 배열의 부분 복사**

```typescript
const numbers = [1, 2, 3, 4, 5];

// 인덱스 1부터 3까지 (3 미포함)
const sliced = numbers.slice(1, 3);
console.log(sliced);  // [2, 3]

// 음수 인덱스 (끝부터)
const last2 = numbers.slice(-2);
console.log(last2);  // [4, 5]

// 원본 배열은 변경되지 않음
console.log(numbers);  // [1, 2, 3, 4, 5]

// 배열 복사
const copy = numbers.slice();
console.log(copy);  // [1, 2, 3, 4, 5]
```

**concat: 배열 연결**

```typescript
const arr1 = [1, 2, 3];
const arr2 = [4, 5, 6];

const combined = arr1.concat(arr2);
console.log(combined);  // [1, 2, 3, 4, 5, 6]

// 여러 배열 연결
const all = arr1.concat(arr2, [7, 8]);
console.log(all);  // [1, 2, 3, 4, 5, 6, 7, 8]

// 원본 배열은 변경되지 않음
console.log(arr1);  // [1, 2, 3]
console.log(arr2);  // [4, 5, 6]
```

### 2-4. 배열 검사

**every: 모든 요소가 조건을 만족하는지 확인**

```typescript
const numbers = [2, 4, 6, 8];

// 모든 요소가 짝수?
const allEven = numbers.every(n => n % 2 === 0);
console.log(allEven);  // true

const nums2 = [2, 4, 6, 7];
console.log(nums2.every(n => n % 2 === 0));  // false (7이 홀수)
```

**some: 하나 이상의 요소가 조건을 만족하는지 확인**

```typescript
const numbers = [1, 2, 3, 4, 5];

// 5보다 큰 요소가 있는가?
const hasLarge = numbers.some(n => n > 4);
console.log(hasLarge);  // true

const nums2 = [1, 2, 3, 4];
console.log(nums2.some(n => n > 10));  // false
```

## 3. 제네릭을 활용한 고급 배열

### 3-1. 제네릭 함수로 배열 처리

**제네릭**을 사용하면 배열의 타입에 관계없이 동작하는 함수를 만들 수 있습니다:

```typescript
// 배열의 첫 번째 요소를 반환하는 함수
function getFirst<T>(arr: T[]): T {
  return arr[0];
}

console.log(getFirst([1, 2, 3]));        // 1 (T는 number)
console.log(getFirst(["a", "b", "c"])); // "a" (T는 string)

// 타입이 유지됨
const firstNumber: number = getFirst([1, 2, 3]);
const firstString: string = getFirst(["x", "y", "z"]);
```

**배열을 반대로 정렬하는 제네릭 함수:**

```typescript
function reverseArray<T>(arr: T[]): T[] {
  return [...arr].reverse();
}

console.log(reverseArray([1, 2, 3]));        // [3, 2, 1]
console.log(reverseArray(["a", "b", "c"])); // ["c", "b", "a"]
```

**배열 필터링 제네릭 함수:**

```typescript
function filterArray<T>(arr: T[], predicate: (item: T) => boolean): T[] {
  return arr.filter(predicate);
}

const numbers = [1, 2, 3, 4, 5];
const evens = filterArray(numbers, n => n % 2 === 0);
console.log(evens);  // [2, 4]

const words = ["apple", "banana", "app", "band"];
const longWords = filterArray(words, w => w.length > 3);
console.log(longWords);  // ["apple", "banana"]
```

### 3-2. 제약이 있는 제네릭 (Constraints)

제네릭에 제약을 두어 특정 조건을 만족하는 타입만 받도록 할 수 있습니다:

```typescript
// T는 반드시 string 타입이어야 함
function toUpperCase<T extends string>(arr: T[]): string[] {
  return arr.map(item => item.toUpperCase());
}

console.log(toUpperCase(["hello", "world"]));  // ["HELLO", "WORLD"]

// ❌ 숫자 배열은 불가
toUpperCase([1, 2, 3]);  // Error: Type 'number' does not satisfy the constraint 'string'
```

**특정 속성을 가진 객체만 받도록 제약:**

```typescript
interface Named {
  name: string;
}

function getNames<T extends Named>(items: T[]): string[] {
  return items.map(item => item.name);
}

const users = [
  { name: "Alice", age: 25 },
  { name: "Bob", age: 30 }
];

console.log(getNames(users));  // ["Alice", "Bob"]

// ❌ name 속성이 없으면 에러
getNames([{ age: 25 }]);  // Error: Property 'name' is missing
```

### 3-3. 다중 제네릭 타입

여러 개의 제네릭 타입을 조합할 수 있습니다:

```typescript
// 두 배열을 합치는 함수
function merge<T, U>(arr1: T[], arr2: U[]): (T | U)[] {
  return [...arr1, ...arr2];
}

const numbers = [1, 2, 3];
const strings = ["a", "b", "c"];

const merged = merge(numbers, strings);
console.log(merged);  // [1, 2, 3, "a", "b", "c"]
// merged의 타입은 (number | string)[]

// 키-값 쌍을 만드는 함수
function zip<K, V>(keys: K[], values: V[]): [K, V][] {
  return keys.map((key, i) => [key, values[i]]);
}

const names = ["Alice", "Bob", "Charlie"];
const ages = [25, 30, 35];

const pairs = zip(names, ages);
console.log(pairs);
// [["Alice", 25], ["Bob", 30], ["Charlie", 35]]
```

### 3-4. 배열 타입 유틸리티

TypeScript에서 제공하는 배열 관련 유틸리티 타입들:

```typescript
// 배열 요소 타입 추출
type ArrayElement<T> = T extends (infer E)[] ? E : never;

type NumberArrayElement = ArrayElement<number[]>;  // number
type StringArrayElement = ArrayElement<string[]>;  // string

// 배열의 마지막 요소 타입 구하기
type Last<T extends any[]> = T extends [...infer _, infer L] ? L : never;

type LastNum = Last<[1, 2, 3]>;  // 3
type LastStr = Last<["a", "b", "c"]>;  // "c"

// 배열의 첫 번째 요소 타입 구하기
type First<T extends any[]> = T extends [infer F, ...infer _] ? F : never;

type FirstNum = First<[1, 2, 3]>;  // 1
```

## 4. 실전 예제

### 4-1. 학생 성적 관리

```typescript
interface Student {
  name: string;
  scores: number[];
}

const students: Student[] = [
  { name: "Alice", scores: [85, 90, 88] },
  { name: "Bob", scores: [78, 82, 80] },
  { name: "Charlie", scores: [92, 95, 90] }
];

// 각 학생의 평균 구하기
function getAverages(students: Student[]): { name: string; average: number }[] {
  return students.map(student => ({
    name: student.name,
    average: student.scores.reduce((sum, score) => sum + score, 0) / student.scores.length
  }));
}

const averages = getAverages(students);
console.log(averages);
// [
//   { name: "Alice", average: 87.67 },
//   { name: "Bob", average: 80 },
//   { name: "Charlie", average: 92.33 }
// ]

// 평균이 85 이상인 학생만 필터링
const topStudents = averages.filter(s => s.average >= 85);
console.log(topStudents);
// [
//   { name: "Alice", average: 87.67 },
//   { name: "Charlie", average: 92.33 }
// ]
```

### 4-2. 중복 제거

```typescript
// 배열의 중복 제거
function removeDuplicates<T>(arr: T[]): T[] {
  return [...new Set(arr)];
}

const numbers = [1, 2, 2, 3, 3, 3, 4, 5, 5];
console.log(removeDuplicates(numbers));  // [1, 2, 3, 4, 5]

const words = ["apple", "banana", "apple", "cherry", "banana"];
console.log(removeDuplicates(words));  // ["apple", "banana", "cherry"]
```

### 4-3. 객체 배열에서 특정 값 추출 및 변환

```typescript
interface Product {
  id: number;
  name: string;
  price: number;
  inStock: boolean;
}

const products: Product[] = [
  { id: 1, name: "Laptop", price: 1200, inStock: true },
  { id: 2, name: "Mouse", price: 30, inStock: false },
  { id: 3, name: "Keyboard", price: 80, inStock: true }
];

// 재고 있는 상품만 필터링하고 이름만 추출
const availableProducts = products
  .filter(p => p.inStock)
  .map(p => p.name);

console.log(availableProducts);  // ["Laptop", "Keyboard"]

// 가격이 100 이상인 상품의 평균 가격
const expensiveAverage = products
  .filter(p => p.price >= 100)
  .reduce((sum, p) => sum + p.price, 0) / 
  products.filter(p => p.price >= 100).length;

console.log(expensiveAverage);  // 1200
```
